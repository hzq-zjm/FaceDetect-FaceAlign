# FaceDetect-FaceAlign
基于ncnn的轻量的、手机端人脸检测和关键点定位工程
# 简介
  对于类似于人脸活体检测的场景，为达到速度与精度的平衡，人脸检测和关键点定位模型大小均约1M。  
  检测速度：高通骁龙835-4核，可达到40FPS
